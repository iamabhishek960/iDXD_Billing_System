# iDXD - Billing System

iDXD Billing System is a simple and efficient web-based billing application that allows users to generate invoices anytime, anywhere using a mobile device or computer.

The system provides an easy-to-use interface for adding customer details, managing items, and automatically calculating totals. Users can edit or delete items, print bills, and export invoices as Excel files for record keeping.

Key Features:
- 📄 Create and manage invoices in real-time  
- 👤 Add customer details (Name, Phone, GST)  
- 🧾 Automatic calculation of total amount  
- ✏️ Edit and delete items بسهولة  
- 🖨️ Print-ready bill format  
- 💾 Export invoices to Excel  
- 📱 Responsive design (works on mobile & desktop)  

This project is designed to make billing fast, accessible, and user-friendly for small businesses and personal use.
